MedMinute AI — Easy Setup

Most of the setup is now automated.

Mac: Double-click SETUP_MEDMINUTE_AI.command.
Windows: Double-click SETUP_MEDMINUTE_AI_WINDOWS.bat.

The setup checks Node.js, installs everything, asks for your private API key, creates the configuration file, starts MedMinute and opens the AI Tutor.

You must personally create and paste your own OpenAI API key. It cannot be created by someone else and should never be shared.

After the first setup, use RUN_MEDMINUTE.command on Mac or RUN_MEDMINUTE_WINDOWS.bat on Windows.
